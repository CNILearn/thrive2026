## C# 14 and .NET 10 - What' s New?

9.6.2026, 14:00 - 15:00, VIP

Dive into the latest innovations across the .NET platform with an accelerated tour of C# 14 and .NET 10. This session highlights the language enhancements that further reduce boilerplate and improve expressiveness (extension members, enhancements for meta-programming), along with runtime and SDK advances, as well as library enhancements such as OpenAPI 3.1 support, enhancements with Blazor, and .NET runtime libraries.

## How Aspire helps developing all .NET solutions (and more)

10.6.2026 - 9:00 - 10:00, Marka Tulija Cicera

If the solution is big or small - Aspire can help. In this session you learn the advantages using Aspire by starting with a small ASP.NET Core application accessing a database. You'll learn how easy it is to convert an existing .NET application to Aspire, and see the advantages you get. After this you'll see using different Aspire integrations used, such as caching, AI, and accessing external services. You'll see the Aspire dashboard in action to monitor and resolve issues with the solution.

[Slides](slides/Aspire-2026.pdf)

### [Samples](aspire)

- Aspireify Codebreaker
- Aspire with HTML frontend
- Aspire with WPF and Windows Forms front-ends
- Aspire starter project with ASP.NET Core Blazor and minimal APIs
- Aspire starter project with Express/React and TypeScript

## How Copilot can help development with Visual Studio 2026

10.6.2026, 16:30 - 17:30, Seneka

In 60 minutes see how GitHub Copilot elevates productivity inside Visual Studio 2026: deeper inline completions, multi-file refactoring suggestions, AI test and benchmark generation, conversational fix explanations, secure prompt context scoping, pull request and diff annotations, and AI-assisted debugging (exception trace summarization, log pattern extraction). Learn prompt patterns that yield maintainable code, governance controls (policy, telemetry, license filtering), customizing Copilot via extension points, and integrating with existing CI quality gates.
